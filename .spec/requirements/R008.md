# R008 — 安全与加密

**Date**: 2026-07-17
**Priority**: P0 (MVP)
**Source**: docs/需求功能文档.md §3.11, D002

## Background
SamSSH 需要保障 SSH 连接安全和凭据存储安全，使用 Windows DPAPI 加密和主机密钥验证。

## Goals
- DPAPI 凭据加密
- 主机密钥验证防 MITM 攻击

## Business Rules

### BR-1001: DPAPI 凭据加密
使用 Windows DPAPI CryptProtectData 加密保存的密码和密钥。F-1001

### BR-1002: 主机密钥验证
防中间人攻击的主机密钥验证机制。F-1003
