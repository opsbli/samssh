# R006 — 主题系统

**Date**: 2026-07-17
**Priority**: P0 (MVP)
**Source**: docs/需求功能文档.md §3.6, D002

## Background
SamSSH 支持 UI 主题（.thm 文件）和终端配色（WezTerm 兼容 TOML）双主题系统。

## Goals
- UI 主题系统（.thm 文件格式）
- 终端颜色主题（WezTerm TOML 格式）
- 主题加载、应用、重载

## Business Rules

### BR-501: UI 主题系统
.thm 文件定义 UI 颜色（背景、前景、顶栏、强调色等）。F-501

### BR-502: UI 主题加载顺序
可执行目录 → CWD 的 theme/ 和 dist/theme/；回退内置默认值。F-502

### BR-503: UI 主题格式
支持 #RRGGBB 和 r,g,b 格式；支持 # 和 ; 注释。F-503

### BR-504: UI 主题键值
支持 bg/fg/top_bg/top_border/accent/muted 等键。F-504

### BR-505: 终端颜色主题
导入 WezTerm 兼容 TOML 配色方案文件。F-506

### BR-506: 终端主题目录
从 term/ 和 dist/term/ 目录加载 .toml 文件。F-507

### BR-507: 终端主题应用
点击主题卡片即时应用至所有终端（含已存在会话）。F-508

### BR-508: 终端主题重载
新增 TOML 文件后点击 Reload from term/ 刷新列表。F-509
