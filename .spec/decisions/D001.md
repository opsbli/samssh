## Question
gpui 是 Zed 编辑器的 GUI 框架，尚未作为独立 crate 发布。如何引入？

## Answer
用户确认从 GitHub (zed-industries/gpui) 直接引入，作为 git 依赖。

## Impact
- Cargo.toml 中 gpui 以 git 依赖形式引入
- 需使用 nightly Rust 工具链
- gpui-component 已确认从 longbridge/gpui-component 引入（git 依赖）
- 旧版 egui 代码不可直接复用
