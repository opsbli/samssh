# D007 — FileManagerView 布局：双面板（远程+本地）

**Date**: 2026-07-21

## Context
SFTP 文件浏览器的布局方式需要确定。需求文档概述中提到「双面板风格」，但节 5.2 的布局图显示的是单面板远程文件列表 + 操作工具栏。

## Question
- **ID**: Q001
- **Text**: SFTP 文件浏览器的布局方式：需求文档 5.2 节显示的是一个单面板远程文件列表 + 操作工具栏；但文档概述（1.0）中提到的「双面板风格」可能指整个工作区的侧边栏+内容区布局。你想采用哪种方案？
