# R005 — 配置与设置

**Date**: 2026-07-17
**Priority**: P0 (MVP)
**Source**: docs/需求功能文档.md §3.5, D002

## Background
SamSSH 需要多页面设置对话框和 Profile 管理功能。配置通过 JSON + DPAPI 加密持久化。

## Goals
- 多页面设置对话框
- Profile 增删改查
- 字体设置/终端颜色设置
- 配置加密存储

## Business Rules

### BR-401: 设置对话框
多页面设置界面。F-401

### BR-402: Profile 管理
增删改查连接配置文件，从对话框加载至登录。F-402

### BR-403: 字体设置
终端字体选择（仅等宽字体），应用至 UI 选项。F-406

### BR-404: 终端颜色设置
主题卡片浏览、点击即时应用。F-407

### BR-405: 配置加密存储
含凭据的配置以 DPAPI 加密保存。F-409
