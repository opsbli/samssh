# E007 — FileManagerView 设计决策

**Source**: 用户 grill 会话 (2026-07-21)
**Type**: decision
**Location**: spec-grill session, Q001-Q005

## Description

用户通过 spec-grill 盘问确认了 FileManagerView（SFTP 双面板文件浏览器）的 5 项关键设计决策：
1. 布局：双面板（远程+本地），类似 WinSCP 风格
2. 文件列表：表格视图（名称/大小/修改时间/权限）
3. 传输操作：工具栏单方向按钮（上传/下载），触发系统文件对话框
4. SFTP 标签创建：基于已有 SSH 连接复用
5. 本地面板：仅文件列表，无目录树（双击进入目录，↑按钮回上级）

## Relevance

这 5 项决策直接决定了 FileManagerView 的架构设计、组件拆分和 UI 布局方案。
