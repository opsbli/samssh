# R003 — SFTP 文件管理

**Date**: 2026-07-21 (Updated)
**Priority**: P0 (MVP)
**Source**: docs/需求功能文档.md §3.3, D002, D007, D008, D009, D010, D011

## Background
SamSSH 提供内置双面板（远程+本地）SFTP 文件浏览器，基于已有 SSH 连接复用 SFTP 通道。支持表格视图文件浏览、文件操作（上传/下载/重命名/删除/新建目录）和目录导航。

## Goals
- 双面板文件浏览器（远程面板 + 本地面板）
- 表格视图文件列表（名称/大小/修改时间/权限）
- 目录导航（路径栏、前进/后退/上级）
- 文件操作：上传（按钮+系统对话框）、下载（按钮+保存对话框）
- 文件管理：新建目录、重命名、删除（含确认框）

## Business Rules

### BR-201: 双面板文件浏览
双面板布局：左侧本地面板 + 右侧远程面板，中间为操作工具栏。两个面板共用 FileBrowser 组件，数据源分别为本地文件系统和 SFTP 远程服务器。面板大小可拖拽调整。D007, D011

### BR-202: 文件详情展示 — 表格视图
以表格形式展示文件/目录，列：文件名、大小、修改时间、权限。目录优先排序。D008

### BR-203: 目录导航
路径输入框 + 上级按钮(↑) + 刷新按钮(↻)。双击目录进入，↑返回父目录，路径栏可手动输入路径跳转。D011

### BR-204: 目录优先排序
目录排在文件前，按名称字母序（不区分大小写）。

### BR-205: 新建目录
工具栏【新建目录】按钮 → 弹出命名对话框 → 远程创建目录。

### BR-206: 重命名
工具栏【重命名】按钮 → 内联编辑或对话框 → 提交后重命名。

### BR-207: 删除
工具栏【删除】按钮 → 确认对话框（列出待删除项）→ 确认后删除远程文件/目录（递归删除目录）。

### BR-208: 上传文件
工具栏【上传】按钮 → 系统文件对话框（rfd）选择本地文件 → 上传至远程当前目录。

### BR-209: 下载文件
工具栏【下载】按钮（或右键菜单）→ 系统保存对话框（rfd）选择本地路径 → 从远程下载文件。

### BR-210: SFTP 标签创建
用户从已连接的 SSH 会话中（右键菜单或标签操作）选择「打开 SFTP」→ 复用 SSH 连接建立 SFTP 通道 → 创建 SessionKind::SFTP 标签。D010

### Constraints
- 远程操作通过 russh-sftp 异步执行
- 本地操作通过 tokio::fs（异步文件 I/O）
- 上传/下载进度通过 TransferTask 状态跟踪
- 删除操作需要二次确认
