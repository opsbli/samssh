# D006 — 新增 R010 映射新建连接对话框

**Date**: 2026-07-20
**Status**: approved

## Context

现有的 R001~R009 覆盖了 SSH 连接管理、终端、SFTP、侧边栏等核心功能域，但"新建连接对话框"和"右键菜单删除"作为用户交互入口，没有专门的 Requirement 映射。

## Evidence

- E006: 代码已实现 dialog.rs、workspace.rs 中的新建连接对话框和右键菜单
- 现有 R001~R009 中无专门映射该功能的条目

## Question

新建连接对话框和右键菜单功能应映射到哪个 Requirement？

## Answer

新增 R010「新建/编辑连接对话框」，专门映射该功能。

## Impact

- 新增 `.spec/requirements/R010.md`
- 更新 `app_state.puml` 添加 ProfileDialog 和 Workspace 实体
- 更新 `architecture.puml` 添加 ProfileDialog 组件
- 更新 `module_deps.md` 更新 dialog.rs 为 "新建/编辑连接对话框 (R010)"
- 更新追溯矩阵和 design-to-requirement 映射
