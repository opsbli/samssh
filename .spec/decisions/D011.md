# D011 — 本地面板：仅文件列表，无目录树（MVP 简化）

**Date**: 2026-07-21

## Context
本地面板的展示范围需要确定。双面板布局中，左侧本地面板是否需要独立的目录树导航。

## Evidence
- **ID**: E007
- **Summary**: 双面板布局确认后，本地面板的范围需要定义

## Question
- **ID**: Q005
- **Text**: 本地面板的展示范围：左边的本地面板需要目录树导航，还是只显示一个目录的文件列表？（点击目录名进入子目录）
  - 选项 A：仅文件列表 — 显示当前目录内容，双击目录进入，↑按钮回上级，路径栏跳转
  - 选项 B：目录树 + 文件列表 — 类似 Windows 资源管理器

## Answer
用户选择了**选项 A：仅文件列表**，并确认双击目录进入、↑按钮回上级、路径栏跳转的理解正确。
