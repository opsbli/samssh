# D010 — SFTP 标签创建：基于已有 SSH 连接复用

**Date**: 2026-07-21

## Context
SFTP 标签页的创建方式需要确定。现有 session.rs 只支持 Terminal 类型标签的创建。

## Evidence
- **ID**: E007
- **Summary**: 需要定义用户如何打开 SFTP 文件浏览器标签

## Question
- **ID**: Q004
- **Text**: SFTP 标签页的创建方式：用户如何打开一个 SFTP 文件浏览器标签？
  - 选项 A：基于已有 SSH 连接复用 — 先 SSH 连接，再通过操作「打开 SFTP」复用已有 SSH 连接
  - 选项 B：直接从 Profile 创建 SFTP 标签，跳过终端

## Answer
用户选择了**选项 A：基于已有 SSH 连接复用**。
